🚀 Presenting: Automotive Sales Analytics Dashboard

I’m excited to share my recent project where I built an interactive Power BI dashboard to analyze and visualize automobile sales data. The goal was to transform raw data into meaningful insights through effective and visually appealing data storytelling.

🔍 Project Overview:
This dashboard provides a comprehensive view of car sales performance, helping users explore trends, compare brands, and understand market behavior across different dimensions.

📊 Key Features:
• Brand-wise sales analysis using interactive visuals
• Year-over-year sales trend analysis
• Comparison of different car segments and categories
• Identification of top-performing brands and models
• Interactive filters (Year, Brand, Category)
• Multi-page dashboard for both overview and detailed insights

🛠️ Tools & Skills Applied:
• Power BI (Visualizations, DAX, Slicers, Navigation, Interactions)
• Data Cleaning & Transformation
• Dashboard Design & Layout
• Data Analysis & Interpretation

This project helped me understand how powerful data visualization can be in simplifying complex datasets and turning them into actionable insights for decision-making.
